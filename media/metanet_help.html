Input and Options:

The input to the MetaNet webtool is a list of genes in Entrez Gene ID (ex: 27) or Gene Symbol (ex: ABL2) formats, which are entered into the text box marked "Enter genes here:". For a sample gene list, please click the text immediately below the input genes text box. The gene set collection can be changed using a drop down list and gene sets within the selected collected will be tested for enrichment within the input gene list.

The MetaNet algorithm options can be changed via the Options dialog box that is accessible by clicking the "Options" link. Options include the p-value threshold for determining whether a gene set is consider to be enriched or not, which can be changed using a slider or by typing a number in the input text box above the slider. The enrichment p-value threshold can optionally correspond to post Benjamini correction for multiple hypothesis. Likewise, the p-value threshold can be lowered for determining whether an edge should be included from the pre-computed Global Gene Set Meta Networks. The type of Meta Networks to be considerd can be selected through check boxes. Click OK to set changed options and return to main window.

The algorithm runs upon clicking the Submit button.

Enrichment Results:

The enrichment results can be viewed by clicking the "Enrichment" tab. Gene sets are tested for enrichment within the input gene list by computing a p-value using Fisher's Exact Test. The test results are plotted as an interactive column chart using the high charts plugin. The number of genes in the gene set that are also in the input list along with the uncorrected and Benjamini-corrected p-values are displayed on hover over the columns. Further details needed for computing the p-values are displayed in the right side panel when the respective columns are clicked.

The graph can be printed or saved in one of PNG, JPEG, PDF, or SVG formats by clicking the respective buttons in the top-right corner of the graph. The results can also be downloded as a table by clicking the "Download Enrichment Results" link in top section of the right side panel.

Note that clicking any selected tab again will hide the contents and can be re-displayed by clicking the tab.

Meta Network:

The enriched gene sets are connected if there are functional relationships between them as determined via pre-computed Meta Networks. The current implementation of MetaNet supports three types of Meta Networks per gene set collection:

1. Co-membership. Gene sets are connected if there are a significant number of shared gene members. These edges are colored black by default.
2. Co-enrichment: Differential Expression. Gene sets are connected if there are a significant number of published gene expression experiments where both gene sets are enriched while accounting for shared members. These edges are colored green by default.
3. Linkage: Protein-Protein Interaction. Gene sets are connected if there are a significant number of reported physical interactions between the unique members of the two gene sets. These edges are colored purple by default.

In the user-specific Meta Network, each node represents a single gene set enriched in the user-inputted gene list. Diamond shaped nodes are independent gene sets; i.e. those gene sets that have a greater number of input genes that are unique to that gene set than shared with any other enriched gene set. Conversely, circle nodes are dependent gene sets, where they consist of a greater number of input genes that are shared with at least one other enriched gene set. The legend can be displayed by clicking the "Legend" menu item above the network. The legend can be moved by clicking and dragging the top of the box.

Node or edge size, color, or shape can be changed via the respective menu items above the network. The network layout can be re-applied or changed using a different algorithm using the "Layout" menu item. The network image can be saved in PNG or PDF formats and the network data can be saved in GraphML and SIF formats for re-import in other graph visualization programs via the "Save" menu item.

The interactive network is visualized using the Cytoscape Web Plugin. The Cytoscape Web graph visualization controls for zooming and panning are placed in the top right corner in the Meta Network tab but can be moved by clicking and dragging. The individual nodes can be moved by clicking and dragging as well.

Clicking each node displays the number of input genes in the respective gene set with corresponding NCBI Entrez Gene links in the right side panel.
Clicking each edge displays the type of meta edge and the p-value indicating the confidence in the pre-computed relationship between linked gene sets in the right side panel.
